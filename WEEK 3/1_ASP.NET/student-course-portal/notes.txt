1. angular.json
Contains Angular project configuration such as build settings, assets, styles, scripts, and budgets.

2. tsconfig.json
Main TypeScript configuration file for the project.

3. tsconfig.app.json
Contains TypeScript configuration specifically for the Angular application.

4. package.json
Stores project information, dependencies, and npm scripts.

5. src/main.ts
The entry point of the Angular application. It bootstraps the root application.

6. src/app/app.config.ts
Contains application-wide configuration such as providers and routing.

7. src/app/app.ts
The root component of the Angular application.

8. src/index.html
The main HTML page where the Angular application is loaded.

Compiled Application Code:
The compiled Angular application code is generated in the main-*.js file inside the dist/student-course-portal/browser folder.

Budget Configuration:
maximumWarning - Shows a warning if the application bundle size exceeds the specified limit.
maximumError - Fails the production build if the application bundle size exceeds the specified limit.
